# Smartmockups - SPA version

## Getting Started

First, run the development server:

```js
yarn dev
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

To create production build, run:

```js
yarn build
```

To run eslint check:
```js
yarn lint
```

To run eslint autofix:
```js
yarn lint:fix
```

To run test:
```js
yarn test
```

To run stylelint:
```js
yarn lint:styles
```
